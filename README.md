# OS-166-GMSSL
> 基于FreeRTOS on RISC-V 的国密算法（SM3, SM4）实现

## 文件目录结构

下面是对工程文件目录结构的说明：

```
.
├─doc
│  └─pic
├─firmware
├─include
├─lib
├─src
└─test
```

* doc：存放绝大部分文档
  * pic: 文档对应Markdown 文件的图片链接位置
* firmware：已经完成，可用于测试的第三方固件
* include：工程自定义头文件
* lib：工程使用的第三方库
* src：工程主要代码
* test：工程单元测试文件
